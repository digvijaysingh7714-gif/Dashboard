# 🌱 GREEN KHATI - Agriculture Management System

A comprehensive web application for managing agricultural products, built with Python and Streamlit.

## Features

- **Product Management**: Add, edit, delete, and view products with image support
- **Sales Management**: Record sales, auto-update inventory, generate bills
- **Purchase Management**: Record purchases from suppliers, auto-update inventory
- **Inventory Management**: Real-time stock tracking, low-stock alerts
- **Dashboard**: Overview with charts and summary cards
- **User Roles**: Admin (full access) and Customer (view & purchase)

## Setup

1. Install dependencies:
   ```
   
   pip install -r requirements.txt
   ```

2. Run the application:
   ```
   streamlit run app.py
   ```
